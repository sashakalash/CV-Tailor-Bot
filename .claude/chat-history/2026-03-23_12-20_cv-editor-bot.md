# CV Editor Telegram Bot — Initial Build

**Date**: 2026-03-23

## Goal

Build a Telegram bot that takes a PDF CV + job description, uses Claude AI to reorganize/highlight relevant skills (without fabricating new ones), and sends back a tailored PDF.

## Key Decisions

- **Telegram bot over web app** — simpler UX, no frontend framework overhead, native file handling
- **TypeScript** (over Python) — user's learning goal, despite existing Python bot in `telegram-vacansies-collector`
- **grammY** — TS-first bot framework; `@grammyjs/conversations` plugin handles multi-step flow as a single async function (no manual state machine)
- **Two-step Claude pipeline** — Haiku extracts CV structure into JSON, Sonnet rewrites targeting the JD. Separated for debuggability and cost efficiency
- **pdfkit** (over @react-pdf/renderer) — imperative, no React runtime overhead for a Node.js bot
- **pdf-parse v2** — class-based API (`new PDFParse({ data })` → `.getText()`)
- **zod v4** — runtime validation of Claude's JSON output (import from `zod/v4`)
- **No database** — in-memory state via grammY conversations plugin, personal tool

## What Was Done

### Files Created (12 source files)

- `src/index.ts` — entry point, bot startup, graceful shutdown
- `src/bot.ts` — bot setup, conversations plugin, /start and /cancel commands
- `src/config.ts` — env var validation (BOT_TOKEN, ANTHROPIC_API_KEY)
- `src/types/context.ts` — grammY context type with ConversationFlavor
- `src/types/cv.ts` — Zod schema + TS types for CvData (central contract)
- `src/conversations/cv-edit.ts` — full conversation flow: PDF → extract → parse → JD → rewrite → generate PDF → send
- `src/services/pdf-extractor.ts` — pdf-parse v2 wrapper
- `src/services/cv-analyzer.ts` — Claude API calls (Haiku extract + Sonnet rewrite) with zod validation + retry
- `src/services/pdf-generator.ts` — pdfkit CV template (clean single-column A4)
- `src/prompts/extract.ts` — Haiku system prompt for CV structure extraction
- `src/prompts/rewrite.ts` — Sonnet system prompt for JD-targeted rewriting
- `src/utils/file.ts` — Telegram file download helper
